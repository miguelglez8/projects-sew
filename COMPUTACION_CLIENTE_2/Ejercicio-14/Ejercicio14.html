Almacenamiento de información persistente

¿ En qué consiste la aplicación ?

El usuario debe introducir datos de colegiados (los datos y el formato se indicarán en la memoria), que irán separados con ";" en un .txt (los usuarios deben de ser "APTOS"), luego después de hacer este fichero de texto manualmente, el usuario debe seleccionar el archivo de texto desde nuestra máquina cliente, los datos de los colegiados aptos se almacenarán en una base de datos y se mostrará la información por pantalla haciendo uso de la API File y la Api IndexerDb. Si el usuario lo desea, puede hacer la pantalla completa.
